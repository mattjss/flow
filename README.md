# Flow

A 30×30 vector field evolving through a continuously shifting curl noise pattern. When your cursor enters, a live vortex warps the surrounding arrows into orbital alignment around you. Particles spawn at the cursor and trace real-time paths through the live field. Click to radiate a ring distortion outward that bends the entire field as it passes.

**Interaction:** Move cursor to create a live vortex · Click to send a ring distortion  
**Tech:** Canvas 2D, curl noise, vortex field blending, real-time particle traces
